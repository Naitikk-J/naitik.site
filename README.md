# Naitik — Portfolio

Personal portfolio showcasing AI-powered systems, scalable products, and engineering craft.

## Tech Stack

- **React** + **TypeScript** with Vite
- **Tailwind CSS** + **shadcn/ui**
- **Framer Motion** for animations
- **Lenis** for smooth scrolling

## Getting Started

```bash
npm install
npm run dev
```
